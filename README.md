# PrivateVault

PrivateVault is an iOS app for securely storing private user data.

## Features:
- [x] PIN-style passcode protection
- [x] File gallery
- [x] File search 
- [x] Tags
- [x] iPad drag-and-drop
- [ ] Touch ID 
- [ ] Face ID
