# Features:
- [x] PIN-style passcode protection
- [x] File gallery
- [x] File search 
- [x] Tags
- [x] iPad drag-and-drop
- [x] Import files and photos from device
- [x] Built-in camera
- [x] Document scanner
- [x] Touch ID 
- [x] Face ID
